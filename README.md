# Trumpbill

Meme board / imageboard styled Pump.fun memecoin site.

- **Name:** Trumpbill
- **Ticker:** $Trumpbill

## Run

Open `index.html` or:

```bash
npx serve .
```

## Links

Edit `script.js`:

- `LINKS.buyPrimary` / `buyAlt` — PumpSwap URLs
- `LINKS.dexscreenerEmbed` / `dexscreenerPage` — Dexscreener Solana URLs
